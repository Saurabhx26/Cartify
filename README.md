# 🛒 Cartify - E-Commerce Web Application

## 📌 About the Project
Cartify is a full-stack eCommerce web application designed to provide a smooth and modern online shopping experience.  
Users can browse products, explore categories, and view detailed product information with ease.  
The platform includes essential features like user authentication, cart management, and order placement.  
An admin dashboard is implemented to manage products, categories, users, and orders efficiently.  
The application focuses on clean UI/UX design and responsive layouts for better usability.  
Built using Java and Spring Boot, Cartify ensures a scalable and robust backend architecture.  

---

## 🚀 Features
- 🔐 User Registration & Login (Spring Security)  
- 🛍️ Product Browsing & Category Filtering  
- 🛒 Add to Cart & Order Management  
- 👨‍💼 Admin Dashboard (Manage Products, Categories, Users)  
- 📱 Responsive UI Design  
- ⚡ Fast & Secure Backend  

---

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot  
- **Frontend:** Thymeleaf, HTML, CSS, Bootstrap  
- **Database:** MySQL  
- **Tools:** Git, GitHub, Maven
